MARGINS

Margins are useful to align contents in a way so that they dont crash and burn with other elements of the page this allows them to have an element to have its own area. Margins can be set using pixels, percent of a screen, or set a directional margin to a specific value.

Margin is the invisible space outside of the body content border. So whatever is added to the margin is just some extra space being added around the border of the body of the content. Padding however is the invisble space between the border of the content and the content itself. That way adding more padding will push the content inside giving some boundary from the border of the content body.

This box has been given margin using the basic margin option which allocates the same amount of px, here 25, all around the box. But if you set the left and right directional margins to be auto the box will be centered on every single screen.

POSITION

Position is used to allocate an element to a position relative to some index. There are differnet to positions that can be given to an element.

These include:
Static - default acts like a normal element
Fixed - stays in the position of the view port or stays in the same location the whole time. you can assign locations.
Relative - this is offset from a selected coordinate which here is the top left corner of the box by whatever amount you select. This can also make it mask other elements on the page. Such as a relative box can float over a paragraph covering it.
Absolute - this makes the box like a ghost visible to pass through and other elements will act as if this box is not there giving it the ability to be masked. This makes it absolute to a certain coordinate which can also be a parent element or anything with a static position
Sticky - sticks to the edge that is being scrolled past so top sticky box stays in frame when scrolling past above it and vice versa with bottom. It will snap back to original coordinate when you scroll past the original position again.
